# runway

A place to check out ML models.

## Setup

You'll need to install:

- rust and the tauri-cli
- mongodb
- node and npm
- opencv, llvm, and pkg-config (https://github.com/twistedfall/opencv-rust/blob/master/INSTALL.md)

## Developing

For dev mode, run:

```bash
npm run tauri dev
```

## Building

For prod:

```bash
npm run tauri build
```
